about WeatherX

Go-Live Date: September 1st, 2020

WeatherX:

A Web App, get your current and forecasted weather.

Server is ...

a Node/express.js server, with a PostgreSQL database, Sequelize data modeling,...

uses 2 API sources, TomTom and DarkSky, to get weather data for any place on Earth.

...provides data to the front-end.

The frontend ...

is built mobile-first, using react.js web application framework

with React-Router and Server-Side-Rendering
